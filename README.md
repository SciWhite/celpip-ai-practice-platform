# CELPIP AI Practice Platform

[cel-pip.com](https://cel-pip.com) is a CELPIP preparation platform focused on practical mock practice, AI-assisted writing and speaking feedback, and CLB-related study tools.

## Live site

Visit: https://cel-pip.com

## What it offers

- CELPIP-style mock exam practice
- AI-assisted writing evaluation
- AI-assisted speaking evaluation
- CLB score conversion and immigration language tools
- Reading, listening, writing, and speaking practice resources
- Account-based practice and progress workflows

## Useful entry points

- Main platform: https://cel-pip.com
- Mock exam practice: https://cel-pip.com/mock-exam.html
- CELPIP to CLB converter: https://cel-pip.com/tools/celpip-to-clb-converter/
- Support: https://cel-pip.com/support.html
